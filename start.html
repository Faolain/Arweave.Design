What is Arweave.Design?

Arweave.Design is a webapp that allows you to build your own website easily and upload it onto the Arweave permanent blockchain

Arweave.Design is built ontop of the Grape.js open source webbuilder framework and it is used under the BSD 3-Clause licence which can be found here

What is Arweave?

Arweave is a blockchain file storage system that allows you to store files permanantly and with no ongoing costs. For more information go to Arweave.org
